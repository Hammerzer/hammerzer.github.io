一、基本元器件介绍

Multisim来源于软仓

1、二极管

定义：二极管就是半导体材料被封装之后，在PN结两端加上两个正负极引线制作而成。

特性：单向导通性【电流只能正极到负极】、伏安特性【电流与电压的关系】

1.1 普通二极管特性测试

Multisim 中不认为电源的负极为0电平，故需要负极接地

特性：

  • 二极管导通之后，它所分的电压值为0.7v【工业标准认为】；
  • 发光二极管导通之后所分的电压为1~2v，电流值为5~20mA；
    • 也就是说，发光二极管的发光条件是电流为5~20mA【超过时可能被烧毁】
    • 为使得发光二极管发光，应调整电阻阻值

  • 当二极管两端电压过小时，它是不导通的；当电压值达到0.5v之后，开始导通

  • 二极管反向不导通，当电压达到 击穿电压 时，二极管导通


1.2 稳压二极管

能够稳定电压的二极管【如：ZENER-ZPD5.1 | 稳压5.1v】

实行稳压的条件：

  1. 工作在反向击穿状态【反接】
  2. 反向电压应大于稳压电压


1.2.1 测试

从伏安特性曲线可知，稳压二极管的电流应满足 Izmin < I < Izmax ，且大概为4mA~8mA

已知发光二极管导通电压为 1.66v，设导通电流为10mA，稳压二极管路端电压为5.1v，可以得到的：R2的阻值应为 R2 = ( UZPD5.1 - ULED1 ) / ILED1 = 344Ω

为使得稳压二极管正常工作，取稳压二极管电流为6mA，则流过串联电阻R1的电流为 IR1 = ILED1 + IZPD5.1 = 16mA，电压值为：总电压 - UZPD5.1 = 6.9V，得到串联电阻阻值为431.25Ω ≈ 431Ω

总结

  • 稳压二极管的使用形式为稳压二极管和电阻进行串联
  • 稳压二极管要达到比较好的稳压效果，一定要注意稳压电流的选取
  • 在电路应用中一定要注意串联电阻阻值的选择

1.2.2 两种稳压电路

对于第一种电路：

  • 当电压小于3v 时，未达到击穿电压，其反接在电路中相当于断开，故示波器输出的是R4上的电压
  • 当电压≥3v 时，达到击穿电压，故示波器输出是等于稳压二极管上的电压

对于第二种电路：

  • 当电压小于3V 时，未达到击穿电压，其反接在电路中相当于断开，故示波器输出为0；
  • 当电压≥3V 时，达到击穿电压，稳压二极管分压3V，其余电压负载在R5，示波器输出R5的电压

1.3 整流二极管

用于把交流电变为脉动直流电


1.4 开关二极管

它是电路上为进行【开 | 关】作用而设计的二极管。它由【导通变为截止 | 截止变为导通】所需的时间比一般二极管短

应用：在电路中主要防止反向电流烧坏一些精密器件，起保护作用。


1.5 电容

它有两个电极板，和中间所夹的介质封装而成，具有特定功能的电子器件。

作用：旁路、去耦、滤波、储能


1.5.1 旁路

VCC电源可能会因为外界干扰，如变为蓝色信号，这样的信号对于处理器来说可能会导致错误的逻辑输出。

若电压本身就是波动的，电容可以储能，当外界信号变化过快时，及时稳定信号

故加入电容：

  • 使输入电压均匀化，减小噪声对后级的影响；
  • 进行储能，当外界信号变化过快时，及时进行电压的补偿

1.5.2 去耦电容（退耦）

  • 去耦电容和旁路电容的作用差不多，都有滤除干扰信号的作用，只是旁路电容针对的是输入信号，而去耦电容针对的是输出信号；
  • 去耦电容一般比较大【10uF 或更大】，旁路电容一般根据谐振频率是 0.1uF 或 0.01uF

1.5.3 滤波和储能

滤波作用：滤波杂波，大电容【如旁路电容中的4.7uF】滤低频，小电容【0.1uF 或 0.01uF】滤高频

储能作用：收集电荷

应用：如时钟芯片DS1302供电【在芯片和地直接接储能电容，当无VCC时，可供芯片短暂工作】


1.5.4 电容在电路中的连接

常用电容极性判断：

  • 铝电容：长脚为正极，短脚为负极，或者电容上标有银色负号的一边为负极
  • 瓷片电容和独石电容无极性，但设备生产中有工艺要求

注意：耐压值、防止短接【会爆炸】


1.5.5 电容应用总结
  • 电源上的电容作用一般时滤除电源电压的波动；还可提供一定的电压储备，已备后续电路的需要；
  • 对于一些强干扰的环境，电容的加入可以减少很多电路控制上不必要的麻烦，在使用电容时，需注意耐压值和反接问题
  • 电容使用的取值问题可以参考已有电路，多为工程经验

2、晶体三极管

2.1 三极管介绍

由半导体组成的具有三个电极的晶体管

它是一种利用输入电流控制输出电流的电流控制型器件。【使用基极电流控制发射极电流】

三种工作状态：放大状态、饱和状态、截止状态


2.1.1 放大状态

对于NPN型三极管，基极电流 Ib = βIc，其中β为放大量【对于某个特定三极管，β固定】

基极信号和集电极信号同时汇集给发射极，Ie = Ib + Ic

条件：发射结正偏【Ub > Ue】，集电结反偏【Ub < Uc】


2.1.2 截止状态

条件：发射结反偏【Ub < Ue】，或两端电压为0【Ub = Ue】


2.1.3 饱和状态

饱和状态即，基极电流和集电极电流都很大时，发射极电流已经不受基极控制了

条件：发射结正偏【Ub < Ue】，集电结**正偏**【Ub > Uc】


2.2 三极管三种工作状态测试

其中，XMM1-2-3分别测发射极Ie、基极Ib、集电极Ic的电流，XMM4测基极与发射极之间的电压Ube，XMM5 测发射极与集电极之间的电压Uce

对于共发射极输入特性曲线，当Uce = 0V时，即V2电压为0V，当 Ube < 0.2V时，基极电流Ib = 0A，三极管处于截止状态【如上图，基极电流极小，Ube ≈ 电源电压】

由共发射极输入特性曲线，当Ube > 0.7V时，Ib导通【修改V1为1V】

结论一：当基极和发射极完全导通时，所分得的电压值为0.7V【工程规定】


放大状态测试：修改V2 = 7v

此时，万用表参数如下图所示。此时，Ube = 663mV【发射结正偏】，Uce = 714mV，因为发射极接地，故有Ub < Uc 【集电结反偏】。三极管处于放大状态。

由图可知，Ib = 33uA，Ic = 3.143mA，Ie = 3.176mA，可证实Ie = Ib + Ic，且 Ib = βIc，β ≈ 95

测试Ⅰ：逐渐增大V2【如8V、9V】，发现集电极电流Ic的增长很小

测试Ⅱ：通过减小R1来增大基极电流Ib，很明显集电极电流Ic的增长很明显


由上述测试和三极管输出特性曲线可得

结论二：当基极电流Ib固定，改变Uce【即改变V2电压】，集电极电流Ic基本保持不变

结论三：当Uce固定【即V2电压固定】，增大基极电流Ib，集电极电流Ic也相应增大。但增大受到一些限制，如Vce【即V2】、R2，这些因素均是会影响集电极电流Ic的因素。在测试中，减小R2阻值，也会释集电极电流Ic增大的潜力。


在饱和区，基极电流Ib和集电极电流Ic都很大，要求发射结和集电结均正偏

改变参数：为增大基极电流Ib，修改R1=100Ω，V1=1v

此时，发射结和集电结均正偏，此时 集电极电流Ic ≠ β*基极电流Ib

结论四：饱和区，集电极电流Ic不受基极电流Ib控制且都很大，并且Uce所占电压很小【这种接地法，Ub一直为0.7v】


2.3 小结

2.3.1 截止状态

特点：Ib = 0，Ic = 0， Ie = Ib + Ic = 0

工作状态：集电极和发射极之间相当于开路


2.3.2 放大状态

其中，Ube为0.7V


2.3.3 饱和状态

特点：Ib和Ic 都很大，Ic 不受Ib控制，Uce两端电压很小，相当于导线，工程上认为硅饱和导通的Uce的压降为0.3V，锗管为0.1V。


2.4 三极管主要参数

  • 共发射极电流放大系数：三极管的放大系数一般为10-100倍，应用中一般取30-80为宜
  • 集电极最大允许电流Icm：超过最大允许电流可能会烧毁
  • 集电极最大允许耗散功率Pcm
  • 集电极-发射极间反向击穿电压V（BR）CEO：即Uc或Ue大于Ub时，压差不用大于击穿电压

特征频率：只能方法150MHZ以内的信号


3、场效应管

3.1 场效应管介绍

它时一种电压控电流的器件。它有三个电极：栅极G、漏极D、源极S。


3.2 场效应管三种工作状态测试

如上图，X4与X5其实是测试V1与V2的电压，X1、X2、X3分别测试栅极、漏极、源极电流

  • 当Uds = 10V【漏极与源极间的电压，即V2】时，当Ugs < 1.5V【栅极与源极间的电压，即V1】时，漏极电流极小，可视为不导通。【通常将1.5V称为门限点电压，Uth】
  • 当Uds很小且不变，Ugs增加会导致Id【漏极电流】增大
  • 当Ugs不变，Uds增大，漏极电流Id变化很小

3.2.1 截止区

当Ugs电压小于1.5V的开启电压时，mos管不导通。


3.2.2 可变电阻区

当Uds很小且不变，Ugs增加会导致Id【漏极电流】增大


3.2.3 恒流区

当Ugs不变，Uds增大，漏极电流Id变化很小

过损耗区：当电流和电压都很大，但未击穿时，该器件的功率较大，需要添加散热器避免损坏


3.2.4 击穿区

当Uds很大时【大于击穿电压】，场效应管被击穿。这种情况下，如没有限流电阻，管子很容易烧毁


3.3 三极管和场效应管的比较

三极管：电流控制器件【基极电流控制晶体管导电能力】

场效应管：电压控制器件【用电压产生的电场来控制器件导电能力】

三极管场效应管
输入阻抗不高极高
噪声高小
反应速度较快【兆赫兹级】慢

输入阻抗高时，仅需要极小的中间电流，就可接收前置电路的信号


3.4 结构分类

其中，N沟道MOS的增强型，相当于NPN型三极管；P沟道MOS的增强型，相当于PNP型三极管

耗尽型：当Ugs = 0V时，但Uds ≠ 0V，也能产生漏极电流

增强型：当Ugs < 1.5V，无论Uds，MOS管不导通


4、光耦

实现一种“电-光-电”的转换。

4.1 光耦测试

光耦器件基极电流为16-25mA，此处取16mA

发光二极管点亮，需要5-20mA，光耦器件集电极最大电流10mA，此处取6mA

其中，由上图可知，光耦器件的前置电路占电压1.214V，为使X1测得前置电路电流为16mA，则R1负载电压为 U1 = 5V - 1.214V = 3.786V，故 R1 = 3.786V / 0.016A = 236.625Ω

对于后置电路，一般认为光耦器件处于饱和状态，所分电压一般为0.3V；发光二极管所分电压为1.66V；电流取6mA。综上可得，R2所分电压为 U2 = 5V - 1.66V - 0.3V = 3.04V，R2 = U2 / 0.006A = 506.66Ω

修正R1/R2后

4.2 光耦使用注意事项

光耦电路两侧的供电电源【VCC与VDD】最好不同

8、瞬态电压抑制器【TVS】

它是一种二极管形式的高效能的保护器件，防止瞬态高能量冲击，保护精密器件免受各种浪涌脉冲的损坏。

8.1 TVS作用

  • 在信号和电源线上，能防止微处理器，人体静电、交流浪涌或噪声，导致处理器的失灵
  • 能释放超过10000V，60A以上的脉冲，并能持续10ms，而一般的TTL器件，遇到30ms的10V脉冲时，便会导致损坏，所以利用TVS是既可以防止器件损坏，也可以防止总线之间开关引起的干扰
  • 将TVS放置在信号线和地之间，能避免数据及控制总线受到不必要噪声的影响

二、基本电路验证与分析

1、基本放大电路

1.1 基本共射级放大电路

左侧es信号发生器为输入信号，右侧U0为输出信号

C1与C2为隔直通交作用

电容C1的作用是：防止Ucc对输入信号的影响

Ucc的作用：

  1. 为电路供能
  2. 为电路提供合适的静态工作点【当输入信号反向时，Ube < 0.7V三极管截止，导致输出波形失真】

Rb的作用：为电路提供合适的静态电流Ib

Rc的作用：把放大的电流信号转化为电压信号

各器件的取值范围

1.7 反馈

凡是将放大电路输出端的信号（电压或电流）的一部分或全部引回到输入端，与输入信号叠加，就称为反馈。

负反馈：引回的信号削弱了输入信号。

正反馈：引回的信号增强了输入信号。

其中，基本放大电路放大倍数A【A≥1】，反馈网络放大倍数F【0≤F≤1】

对于闭环放大倍数Af，在深度负反馈条件下，即 1+AF ⋙ 1，即AF ⋙ 0，故 1+AF ≈ AF

此时，闭环放大倍数取决于反馈系数F，而与开环放大倍数A没有关系。通常，反馈网络仅由电阻构成，反馈系数F十分稳定。所以，闭环放大倍数必然相当稳定，诸如温度变化、参数改变、电源电压波动等明显影响开环放大倍数的因素，都不会对闭环放大倍数产生多大影响。


1.7.1 负反馈

负反馈的作用：

  • 稳定静态工作点
  • 稳点放大倍数
  • 改善输入电阻和输出电阻，扩展通频带，改善输出信号波形

负反馈作用举例

例一：

如某放大器的开环放大倍数A=1000，由于外界因素影响，其波动幅值为10%，若反馈系数F为0.009，则闭环放大倍数Af = 1000 / (1 + 1000 * 0.009) = 100。

此时波动10%，变化量仅为1，可见放大倍数的稳定性大大提高

【其实是以降低放大倍数为代价，提高放大倍数的稳点性】

例二：

引入负反馈后，失真了的信号经过反馈网络又送回到输入端，与输入信号反向叠加，得到的净输入信号为正半周小而负半周大。这样正好弥补了放大器的缺陷，使输出信号比较接近正弦波。


1.7.2 正反馈

当|AF| = 1时，即放大系数A和反馈系数F呈倒数关系。此时，电路将产生自激，去除信号源也将产生信号的输出

当|AF| ＞ 1时，放大系数A过大，输出信号将不断增加，如果无及时的限制措施，信号将出现严重失真【如上图】

当|AF| ＜ 1时，反馈系数F过大，信号将不断衰减，直到消失

当正输入端为5V时，LM339输出5V【仿真时出现问题，待补充】，三极管导通，发光二极管发光；


4、功率放大器

功率放大电路是一种以输出较大功率为目的的放大电路。

特点：

  • 输出信号电压大
  • 输出信号电流大
  • 放大电路的输出电阻与负载匹配

电压放大器与功率放大器的区别：

  • 电压放大：不失真地提高输入信号的幅度，以驱动后级功率放大级，通常工作在小信号状态
  • 功率放大：信号不失真或轻度失真的条件下提高输出功率，通常工作在大信号状态

4.1 功率放大电路中三极管的三种工作模式

为什么甲类工作效率低乙类工作效率高?

对于甲类工作电路

对于乙类工作电路，当输入信号处于正半轴，T1导通T2截止；当输入信号处于负半轴，T1截止T2导通

由乙类工作特性图，忽略饱和压降Uces ，其有效电压Vcc，功率P = Vcc*ic


4.2 功率放大器内部的两种结构电路

上为OTL电路

区别：OCL电路无电容，但有双电源供电

  • 当输入信号处于正半轴，T1导通T2截止。此时正电源通过T1给电容充电
  • 当输入信号处于负半轴，T1截止T2导通。此时由电容供电

上为20~20kHZ OCL电路

其中，信号输入放大器正相端【同相】，故放大倍数A = 1 + Rf / R1 = 3倍【符合测试数据】

  • VCC处的大电容和小电容作用为去耦：小电容滤高频，大电容滤低频，且大电容还可以避免VCC供电不及时，由电容及时补充
  • VSS处同理，但要注意电容正极方向，否则会导致电容爆炸
  • 左侧为高通滤波电路，防止直流影响

功率放大器应用：一般功率放大器都会集成于芯片【如LM386】


5、555震荡电路

5.1 555震荡电路

555定时器：一种应用方便的中规模集成电路，广泛应用于信号的产生、变换、控制与检测。因其是有3个5KΩ的电阻分压，故称为555定时器。

复位端：当为零电平时，Q输出0，Q逆输出1，Q逆反相，uo输出为0

复位端一般接VCC，暂无复位功能

5.1.1 多谐振荡器

工作原理：

  • VCC通过R1、R2给电容C充电，刚开始时电容C的电压较小【小于1/3VCC】，即TH/TR < 1/3VCC【工作状态④：uo = 1，三极管截止】
  • 当电容C的充电过程达到1/3VCC < v < 2/3VCC时，即1/3VCC < TH/TR < 2/3VCC【工作状态③：状态不变】
  • 当电容C的充电过程达到v > 2/3VCC时，即TH/TR > 2/3VCC【工作状态②：uo = 0，三极管导通，电容C通过7脚放电端进行放电】
  • 当电容C的放电过程达到1/3VCC < v < 2/3VCC时，即1/3VCC < TH/TR < 2/3VCC【工作状态③：状态不变】
  • 当电容C的放电过程达到v < 1/3VCC时，即TH/TR < 1/3VCC【工作状态④：uo = 1，三极管截止】

震荡周期：

占空比：

  • VCC通过R1及可调电阻上半部分，在经过二极管D1给电容C充电
  • 电容C通过R2及二极管D2、可调电阻下半部分，经过7脚进行放电

5.1.2 设计产生1KHZ占空比可调电路

仿真时出现意外错误【未解决】


5.2 单稳态触发器

工作特性：

  • 具有稳态和暂稳态两个不同的工作状态；
  • 在外界触发脉冲的作用下，能从稳态翻转到暂稳态，在暂稳态维持一段时间后，电路自动返回稳态；
  • 暂稳态不能长久保持，其维持时间取决于电路的自身参数，与触发脉冲无关。

单稳态电路原理：

  • 如上图所示，TR端接ui为高电平的输入波形，在555定时器内部，必然有TR = u+ > u- = 1/3VCC，故C2 = 1

  • VCC通过R给电容C充电，刚开始时电容C的电压较小【小于1/3VCC】，即TH < 1/3VCC【工作状态③：状态不变，即保持状态 ；假设电路通电之前通过复位端进行了复位，则此时输出为uo = 0】【此时，Q逆为1，Q=0，三极管导通，一直处于边充电边放电状态】

  • 突然，ui输入一个低电平下降沿，即TR < 1/3VCC，故C2 = 0；TH端同上，依然有TH < 1/3VCC，即C1 = 1。【工作状态④：uo=1，三极管截止】【此时，VCC给C充电】

  • 当电容C的充电过程达到1/3VCC < v < 2/3VCC时，即1/3VCC < TH < 2/3VCC，故C1 = 1；而TR返回原高电平状态，即TH > 1/3VCC，故C2 = 1；【工作状态③：状态不变，即uo = 1，且Q逆为0，三极管截止】

  • 当电容C的充电过程达到v > 2/3VCC时，即TH > 2/3VCC，故C1 = 0；TR同上，C2 = 1【工作状态②：uo = 0，三极管导通，电容C通过7脚放电端进行放电】【即当v > 2/3VCC时，立即放电使输出uo = 0】

工作特点：

  • 只要给一个低电平的触发信号，那么暂稳态的停留时间为电压0V∼23VCC的充电时间；
  • 充电时间Tp = 1.1RC，故可用于设计定时电路

5.2.1 定时1S电路

接地电容C1的作用：防止外界干扰对555定时器的影响

LED灯用于指示输出为高电平还是低电平

上电路的延时时间Tp = 1.1RC = 1.1 * 91kΩ * 10nF = 0.001s

修改C2 = 10uF，可以使延时时间Tp = 1s



三、实际电路应用

1、多路稳压电源设计

R1、R3的作用：当上方LM7812输出12V时，下方两个并联部分也是12V，故使用R1、R3负担电压和功率

指示灯LED用于通知已通入电源

电路设计要点：

  • 变压器选择：由于7812的最大输入电压范围为14.5V~35V，所以对于变压器可以选择输出12V、2A
  • 整流二极管的选择：【1N4007】能承受1000V的反向电压，平均最大电流为1A，那么它就能保证2A的负载输出【正半轴1A，负半轴1A，和为2A】
  • 2Ω电阻选择：对于5V稳压管，如果输入12V，那么当稳压为5V时，6V的压降，在稳压管上造成的功耗是比较大的，为了减小一些稳压管上的功耗，故选择2Ω电阻分担功耗
  • 稳压管：需要增加散热片，防止稳压管烧毁
  • 输出线上二极管的选择：其作用主要为防止负载上的反相电压，造成对稳压块的影响，故此处选择1N4148【开关二极管 | 反应快，能及时处理】

2、数控恒流电路

运算放大器【同相比例 | 电压跟随器】：对于2.5V的输入，R2处的电压依然为2.5V，该电压压在R2上，故电流为1A

Q1为MOS管

为使得MOS管处于恒流区【MOS管恒流区为5V~15V，一般取大于3V】，故对负载R1有一定要求【如上图：MOS管为3V，R2为2.5V，共VCC = 15V的电源电压，故R1需要负担9.5V，即最大R1 = 9.5V / 1A = 9.5Ω】

VCC电源如果为稳压块输出，其最大电流为1.5A

R2一般阻值小，电流大，故功率大，因此一般选择大功率电阻

  • 限流电阻【R2】要根据需要的电流最大值，选择合适的大功率电阻
  • 对于电源要考虑它能否输出所需的电流
  • 要保证场效应管工作在恒流区

3、峰值检测电路

作用：它用于检测输入信号的峰值电压

电路构成：

  • 电压输入部分【电压跟随器】
  • 检波电路【两个开关管D2D4】
  • 电压保持电路【电容C2】
  • 放电电路【两个三极管】：当基极低电平时，三极管不导通【进行峰值检测】；当基极高电平时，三极管导通，电容C2通过三极管进行放电【此时不能进行峰值检测】

工作原理：

  • 对于5V的交流输入【假设此时输出3V】，电流经过跟随器、两个二极管、后跟随器，最后输出3V信号。此信号反馈到第一个跟随器的负输入端【u- = 3V】【满足虚断，u-端电流为0，即R2不分压，故有u- = 3V】
  • 输入增大【假设到达5V】，跟随器U1输出5V，二极管D2截止；5V电压经过D1、D2给电容C2充电【电容电压5V】，跟随器U2的输出也为5V
  • 当输入信号开始下降【假设下降到2V】，跟随器U1输出2V，而峰值时跟随器U2将5V信号通过R1反馈至D1、D3中间【此时D1管截止】；同样，反馈经R2的电压也为5V【此时D2管导通】，故经过D2流入运算放大器U1

R1的作用：因为二极管D3当两端压差比较大时，会出现有极小的反相电流，会导致电容C2的电压被卸放掉。故经跟随器U2反馈经R1后，通过R1的分压，会减小二极管D3两端压差，进一步防止电容C2向D3反向放电

  • 当输入信号继续下降【假设为-3V】，U1输出-3V，U2的5V反馈经R2、D2到达U1【D1截止】【5V与-3V的压差较大，故R2选择是有要求的】

当R2过小时，峰值检测如上图。当压差较大时，在R2上形成的电流也较大，这个电流来自运算放大器U2，大电流会导致运放U2不稳定

电容C1的作用：破坏输出相位，防止外界干扰导致的自激

  • 当放电电路接地，三极管不导通。跟随器U2输出5V，经R4到达三极管Q1发射极，此时电容C2电压5V，三极管Q1两级没有了压降，进一步阻止C2通过三极管放电

该电路也可用于检测不规则信号的峰值
