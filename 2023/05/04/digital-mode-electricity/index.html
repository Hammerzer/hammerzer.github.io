一、基本元器件介绍

1、二极管

定义：二极管就是半导体材料被封装之后，在PN结两端加上两个正负极引线制作而成。

特性：单向导通性【电流只能正极到负极】、伏安特性【电流与电压的关系】

1.1 普通二极管特性测试

Multisim 中不认为电源的负极为0电平，故需要负极接地

特性：

  • 二极管导通之后，它所分的电压值为0.7v【工业标准认为】；
  • 发光二极管导通之后所分的电压为1~2v，电流值为5~20mA；
    • 也就是说，发光二极管的发光条件是电流为5~20mA【超过时可能被烧毁】
    • 为使得发光二极管发光，应调整电阻阻值

  • 当二极管两端电压过小时，它是不导通的；当电压值达到0.5v之后，开始导通

  • 二极管反向不导通，当电压达到 击穿电压 时，二极管导通


1.2 稳压二极管

能够稳定电压的二极管【如：ZENER-ZPD5.1 | 稳压5.1v】

实行稳压的条件：

  1. 工作在反向击穿状态【反接】
  2. 反向电压应大于稳压电压


1.2.1 测试

从伏安特性曲线可知，稳压二极管的电流应满足 Izmin < I < Izmax ，且大概为4mA~8mA

已知发光二极管导通电压为 1.66v，设导通电流为10mA，稳压二极管路端电压为5.1v，可以得到的：R2的阻值应为 R2 = ( UZPD5.1 - ULED1 ) / ILED1 = 344Ω

为使得稳压二极管正常工作，取稳压二极管电流为6mA，则流过串联电阻R1的电流为 IR1 = ILED1 + IZPD5.1 = 16mA，电压值为：总电压 - UZPD5.1 = 6.9V，得到串联电阻阻值为431.25Ω ≈ 431Ω

总结

  • 稳压二极管的使用形式为稳压二极管和电阻进行串联
  • 稳压二极管要达到比较好的稳压效果，一定要注意稳压电流的选取
  • 在电路应用中一定要注意串联电阻阻值的选择

1.2.2 两种稳压电路

对于第一种电路：

  • 当电压小于3v 时，未达到击穿电压，其反接在电路中相当于断开，故示波器输出的是R4上的电压
  • 当电压≥3v 时，达到击穿电压，故示波器输出是等于稳压二极管上的电压

对于第二种电路：

  • 当电压小于3V 时，未达到击穿电压，其反接在电路中相当于断开，故示波器输出为0；
  • 当电压≥3V 时，达到击穿电压，稳压二极管分压3V，其余电压负载在R5，示波器输出R5的电压

1.3 整流二极管

用于把交流电变为脉动直流电


1.4 开关二极管

它是电路上为进行【开 | 关】作用而设计的二极管。它由【导通变为截止 | 截止变为导通】所需的时间比一般二极管短

应用：在电路中主要防止反向电流烧坏一些精密器件，起保护作用。


1.5 电容

它有两个电极板，和中间所夹的介质封装而成，具有特定功能的电子器件。

作用：旁路、去耦、滤波、储能


1.5.1 旁路

VCC电源可能会因为外界干扰，如变为蓝色信号，这样的信号对于处理器来说可能会导致错误的逻辑输出。

若电压本身就是波动的，电容可以储能，当外界信号变化过快时，及时稳定信号

故加入电容：

  • 使输入电压均匀化，减小噪声对后级的影响；
  • 进行储能，当外界信号变化过快时，及时进行电压的补偿

1.5.2 去耦电容（退耦）

  • 去耦电容和旁路电容的作用差不多，都有滤除干扰信号的作用，只是旁路电容针对的是输入信号，而去耦电容针对的是输出信号；
  • 去耦电容一般比较大【10uF 或更大】，旁路电容一般根据谐振频率是 0.1uF 或 0.01uF

1.5.3 滤波和储能

滤波作用：滤波杂波，大电容【如旁路电容中的4.7uF】滤低频，小电容【0.1uF 或 0.01uF】滤高频

储能作用：收集电荷

应用：如时钟芯片DS1302供电【在芯片和地直接接储能电容，当无VCC时，可供芯片短暂工作】


1.5.4 电容在电路中的连接

常用电容极性判断：

  • 铝电容：长脚为正极，短脚为负极，或者电容上标有银色负号的一边为负极
  • 瓷片电容和独石电容无极性，但设备生产中有工艺要求

注意：耐压值、防止短接【会爆炸】


1.5.5 电容应用总结
  • 电源上的电容作用一般时滤除电源电压的波动；还可提供一定的电压储备，已备后续电路的需要；
  • 对于一些强干扰的环境，电容的加入可以减少很多电路控制上不必要的麻烦，在使用电容时，需注意耐压值和反接问题
  • 电容使用的取值问题可以参考已有电路，多为工程经验

2、晶体三极管

2.1 三极管介绍

由半导体组成的具有三个电极的晶体管

它是一种利用输入电流控制输出电流的电流控制型器件。【使用基极电流控制发射极电流】

三种工作状态：放大状态、饱和状态、截止状态


2.1.1 放大状态

对于NPN型三极管，基极电流 Ib = βIc，其中β为放大量【对于某个特定三极管，β固定】

基极信号和集电极信号同时汇集给发射极，Ie = Ib + Ic

条件：发射结正偏【Ub > Ue】，集电结反偏【Ub < Uc】


2.1.2 截止状态

条件：发射结反偏【Ub < Ue】，或两端电压为0【Ub = Ue】


2.1.3 饱和状态

饱和状态即，基极电流和集电极电流都很大时，发射极电流已经不受基极控制了

条件：发射结正偏【Ub < Ue】，集电结**正偏**【Ub > Uc】


2.2 三极管三种工作状态测试

其中，XMM1-2-3分别测发射极Ie、基极Ib、集电极Ic的电流，XMM4测基极与发射极之间的电压Ube，XMM5 测发射极与集电极之间的电压Uce

对于共发射极输入特性曲线，当Uce = 0V时，即V2电压为0V，当 Ube < 0.2V时，基极电流Ib = 0A，三极管处于截止状态【如上图，基极电流极小，Ube ≈ 电源电压】

由共发射极输入特性曲线，当Ube > 0.7V时，Ib导通【修改V1为1V】

结论一：当基极和发射极完全导通时，所分得的电压值为0.7V【工程规定】


放大状态测试：修改V2 = 7v

此时，万用表参数如下图所示。此时，Ube = 663mV【发射结正偏】，Uce = 714mV，因为发射极接地，故有Ub < Uc 【集电结反偏】。三极管处于放大状态。

由图可知，Ib = 33uA，Ic = 3.143mA，Ie = 3.176mA，可证实Ie = Ib + Ic，且 Ib = βIc，β ≈ 95

测试Ⅰ：逐渐增大V2【如8V、9V】，发现集电极电流Ic的增长很小

测试Ⅱ：通过减小R1来增大基极电流Ib，很明显集电极电流Ic的增长很明显


由上述测试和三极管输出特性曲线可得

结论二：当基极电流Ib固定，改变Uce【即改变V2电压】，集电极电流Ic基本保持不变

结论三：当Uce固定【即V2电压固定】，增大基极电流Ib，集电极电流Ic也相应增大。但增大受到一些限制，如Vce【即V2】、R2，这些因素均是会影响集电极电流Ic的因素。在测试中，减小R2阻值，也会释集电极电流Ic增大的潜力。


在饱和区，基极电流Ib和集电极电流Ic都很大，要求发射结和集电结均正偏

改变参数：为增大基极电流Ib，修改R1=100Ω，V1=1v

此时，发射结和集电结均正偏，此时 集电极电流Ic ≠ β*基极电流Ib

结论四：饱和区，集电极电流Ic不受基极电流Ib控制且都很大，并且Uce所占电压很小【这种接地法，Ub一直为0.7v】


2.3 小结

2.3.1 截止状态

特点：Ib = 0，Ic = 0， Ie = Ib + Ic = 0

工作状态：集电极和发射极之间相当于开路


2.3.2 放大状态

其中，Ube为0.7V


2.3.3 饱和状态

特点：Ib和Ic 都很大，Ic 不受Ib控制，Uce两端电压很小，相当于导线，工程上认为硅饱和导通的Uce的压降为0.3V，锗管为0.1V。


2.4 三极管主要参数

  • 共发射极电流放大系数：三极管的放大系数一般为10-100倍，应用中一般取30-80为宜
  • 集电极最大允许电流Icm：超过最大允许电流可能会烧毁
  • 集电极最大允许耗散功率Pcm
  • 集电极-发射极间反向击穿电压V（BR）CEO：即Uc或Ue大于Ub时，压差不用大于击穿电压

特征频率：只能方法150MHZ以内的信号


3、场效应管

3.1 场效应管介绍

它时一种电压控电流的器件。它有三个电极：栅极G、漏极D、源极S。


3.2 场效应管三种工作状态测试

如上图，X4与X5其实是测试V1与V2的电压，X1、X2、X3分别测试栅极、漏极、源极电流

  • 当Uds = 10V【漏极与源极间的电压，即V2】时，当Ugs < 1.5V【栅极与源极间的电压，即V1】时，漏极电流极小，可视为不导通。【通常将1.5V称为门限点电压，Uth】
  • 当Uds很小且不变，Ugs增加会导致Id【漏极电流】增大
  • 当Ugs不变，Uds增大，漏极电流Id变化很小

3.2.1 截止区

当Ugs电压小于1.5V的开启电压时，mos管不导通。


3.2.2 可变电阻区

当Uds很小且不变，Ugs增加会导致Id【漏极电流】增大


3.2.3 恒流区

当Ugs不变，Uds增大，漏极电流Id变化很小

过损耗区：当电流和电压都很大，但未击穿时，该器件的功率较大，需要添加散热器避免损坏


3.2.4 击穿区

当Uds很大时【大于击穿电压】，场效应管被击穿。这种情况下，如没有限流电阻，管子很容易烧毁


3.3 三极管和场效应管的比较

三极管：电流控制器件【基极电流控制晶体管导电能力】

场效应管：电压控制器件【用电压产生的电场来控制器件导电能力】

三极管场效应管
输入阻抗不高极高
噪声高小
反应速度较快【兆赫兹级】慢

输入阻抗高时，仅需要极小的中间电流，就可接收前置电路的信号


3.4 结构分类

其中，N沟道MOS的增强型，相当于NPN型三极管；P沟道MOS的增强型，相当于PNP型三极管

耗尽型：当Ugs = 0V时，但Uds ≠ 0V，也能产生漏极电流

增强型：当Ugs < 1.5V，无论Uds，MOS管不导通


4、光耦

实现一种“电-光-电”的转换。

4.1 光耦测试

光耦器件基极电流为16-25mA，此处取16mA

发光二极管点亮，需要5-20mA，光耦器件集电极最大电流10mA，此处取6mA

其中，由上图可知，光耦器件的前置电路占电压1.214V，为使X1测得前置电路电流为16mA，则R1负载电压为 U1 = 5V - 1.214V = 3.786V，故 R1 = 3.786V / 0.016A = 236.625Ω

对于后置电路，一般认为光耦器件处于饱和状态，所分电压一般为0.3V；发光二极管所分电压为1.66V；电流取6mA。综上可得，R2所分电压为 U2 = 5V - 1.66V - 0.3V = 3.04V，R2 = U2 / 0.006A = 506.66Ω

修正R1/R2后

4.2 光耦使用注意事项

光耦电路两侧的供电电源【VCC与VDD】最好不同

若两侧信号相同且具有干扰新信号，光耦电路会失去隔离两侧电路的作用

6、继电器

6.1 继电器介绍

它是一种“自动开关”，通过【低电压、小电流】去控制【高电压、大电流】。

常用术语

  • 常开触点：如3、5脚
  • 常闭触点：如3、4脚

应用：对开关速度要求不高的场合

8、瞬态电压抑制器【TVS】

它是一种二极管形式的高效能的保护器件，防止瞬态高能量冲击，保护精密器件免受各种浪涌脉冲的损坏。

8.1 TVS作用

  • 在信号和电源线上，能防止微处理器，人体静电、交流浪涌或噪声，导致处理器的失灵
  • 能释放超过10000V，60A以上的脉冲，并能持续10ms，而一般的TTL器件，遇到30ms的10V脉冲时，便会导致损坏，所以利用TVS是既可以防止器件损坏，也可以防止总线之间开关引起的干扰
  • 将TVS放置在信号线和地之间，能避免数据及控制总线受到不必要噪声的影响

② 减小Rb的后果【如R1=100kΩ】：输出信号的部分负向信号被削减【饱和失真】

这是因为，R1减小，基极电流Ib增大，导致集电极的电流Ic也增大

又因为输出电压Uce = -ic*Rc，故输出信号峰值也增大，但反向电压也反向增大至小于0，使三极管饱和失真。


③ 增大Rb的后果【如R1=1MΩ】：输出信号的部分正向信号被削减【截止失真】


④ 如何确定合适的静态工作点Q：即分析与Rb、VCC、Rc的关系，参考此处

Rb增大，基极电流Ib减小，导致Ic减小，由公式得Uce增大

VCC增大，基极电流Ib增大，导致Ic增大


1.5 温度影响

这种电路虽然结构简单，但最大的缺点是静态工作点不稳点。当环境温度变化、电源电压波动、更换三极管时，都会使原来的静态工作点改变，严重时使放大器不能工作。


1.6、分压偏置式共射极三极管放大电路

B点处的电压Vb可由R1和R2分压计算

1.7 反馈

凡是将放大电路输出端的信号（电压或电流）的一部分或全部引回到输入端，与输入信号叠加，就称为反馈。

负反馈：引回的信号削弱了输入信号。

正反馈：引回的信号增强了输入信号。

其中，基本放大电路放大倍数A【A≥1】，反馈网络放大倍数F【0≤F≤1】

对于闭环放大倍数Af，在深度负反馈条件下，即 1+AF ⋙ 1，即AF ⋙ 0，故 1+AF ≈ AF

此时，闭环放大倍数取决于反馈系数F，而与开环放大倍数A没有关系。通常，反馈网络仅由电阻构成，反馈系数F十分稳定。所以，闭环放大倍数必然相当稳定，诸如温度变化、参数改变、电源电压波动等明显影响开环放大倍数的因素，都不会对闭环放大倍数产生多大影响。


1.7.1 负反馈

负反馈的作用：

  • 稳定静态工作点
  • 稳点放大倍数
  • 改善输入电阻和输出电阻，扩展通频带，改善输出信号波形

负反馈作用举例

例一：

如某放大器的开环放大倍数A=1000，由于外界因素影响，其波动幅值为10%，若反馈系数F为0.009，则闭环放大倍数Af = 1000 / (1 + 1000 * 0.009) = 100。

此时波动10%，变化量仅为1，可见放大倍数的稳定性大大提高

【其实是以降低放大倍数为代价，提高放大倍数的稳点性】

例二：

引入负反馈后，失真了的信号经过反馈网络又送回到输入端，与输入信号反向叠加，得到的净输入信号为正半周小而负半周大。这样正好弥补了放大器的缺陷，使输出信号比较接近正弦波。


1.7.2 正反馈

当|AF| = 1时，即放大系数A和反馈系数F呈倒数关系。此时，电路将产生自激，去除信号源也将产生信号的输出

当|AF| ＞ 1时，放大系数A过大，输出信号将不断增加，如果无及时的限制措施，信号将出现严重失真【如上图】

当|AF| ＜ 1时，反馈系数F过大，信号将不断衰减，直到消失

3、集成运算放大电路

集成运算放大器是一种具有很高放大倍数的多级直接耦合放大电路，是发展最早、应用最广泛的一种模拟集成电路。

特点：高增益、高可靠性、低成本、小尺寸

表现： 【放大倍数A、输入阻抗高r(对前级电路影响小)、输出电阻小(对后级电路影响小)、共模抑制比（抗干扰能力）】

运算放大器特点：

★ 工作在线性区

  • 理想运放两输入端点位相等【虚短】【Uo = Aud * Ui , Ui = Up - Un，故Ui = Uo / Aud，因为Aud很大，故Ui ≈ 0，即Up = Un】【Up为正极电压】
  • 理想运放输入电流等于0【虚断】【理想的输入阻抗高，故正向端与反向端电流为0】

★ 工作在非线性区【依然虚断】
