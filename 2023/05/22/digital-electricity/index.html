1.4 OC门

OC门【Open collector gate】，集电极开路的门电路

TTL门有局限性，比如输出端不能并联、输出高低电压固定不可调节、以及驱动能力低等。而OC门通过外接电源和负载电阻可以克服以上缺点

TTL门输出端为何不能并联？

当A端高电平且B端低电平，即此时Q1、Q4导通，自然会形成电流。而此时Q1高电平处于深度饱和状态，电流会比较大，故这两个三极管容易烧毁。

因此不能并联，为实现目标，电路改进【去除上方的三极管，将Q2集电极悬空，即集电极开路】

此时，如果A端三极管导通，A输出低电平；如果三极管截止，相当于集电极断开，三极管属于高阻态【此时，想要输出高电平，需要上拉电阻和电源，如上图所示，三极管集电极断开，B端相当于由VCC提供高电平】【TTL为5V供电，VCC供电无要求，故它可以为后级电路提供更强的驱动能力，即电平转换功能】

线与功能：由于三极管截止时集电极是断开的，故不会形成电流，可用于实现“线与逻辑”【只有都为高电平，输出才为高电平】

驱动功能：集电极接一个电阻、一个发光二极管、一个5V VCC，如果电阻取值合适，电流经过电阻、三极管到地，这是驱动功能

为什么引入OC门？

实际使用中，有时需要两个或两个以上与非门的输出端连接在同一条导线上，将这些与非门上的数据【状态电平】用同一条导线输送出去。

因此，需要一种新的与非门电路【OC门】来实现“线与逻辑”。

OC门主要用于3个方面： 实现与或非逻辑、用做电平转换、用做驱动器。

由于OC门电路的输出管的集电极悬空，使用时需外接一个上拉电阻Rp到电源VCC。OC门使用上拉电阻以输出高电平；此外为了加大输出引脚的驱动能力，上拉电阻阻值的选择原则，从降低功耗及芯片的灌电流能力考虑应当足够大，从确保足够的驱动电流考虑应当足够小。

OD门：为漏极开路的门电路【漏极 | MOS管 | 即场效应管电路】，功能和OC门差不多

1.6 CMOS传输门

TG【Transmission Gate】：当传输门导通，ui可正常输出uo，且ui = uo；当传输门截止，ui ≠ uo。可用于双向模拟开关

MOS管的漏极和源极结构对称，可互换使用。因此，CMOS传输门的输出端和输入端可互换使用

C、C逆为互补的控制信号

  • 当 C = VDD，ui = 0~VDD时，上下两个MOS管中至少有一管导通，输出与输入ui = uo，称为传输门开通
  • 当 C = 0V，ui = 0~VDD时，上下两个MOS管均截止，输出与输入之间呈现高电阻，相当于断开，称为传输门关闭


1.7 小结

比较：CMOS门电路【MOS管 | 场效应管】 VS TTL门电路【三极管】

  • 【以下为前者的特点】功耗极低
  • 扛干扰能力强
  • 电源电压范围宽
  • 输出信号摆幅大【高电平 ≈ VDD | 低电平 ≈ 0V】【TTL电路低电平约为0.3V】
  • 输入阻抗高【对前级电路影响小】
  • 扇出系数大【带负载能力强】
  • 一般情况下，反应相对于TTL门来说较慢【电平翻转慢】

闲置输入端的处理

  • 与门和与非门：多余的输入端接正电源或与有用输入端并接
  • 或门和或非门：多余的输入端接地或与有用输入端并接
  • CMOS电路：多余的输入端与有用输入端并接【仅适用于工作频率很低的场合】
  • TTL电路输入端悬空相当于输入高电平，CMOS电路多余输入端不允许悬空

七、触发器

数字电路，分为组合逻辑电路和时序逻辑电路。组合逻辑电路的基本单位是门电路，时序逻辑电路的基本单位是触发器

  • 门电路某一时刻的输出信号，完全取决于该时刻的输入信号，无记忆功能
  • 触发器具有记忆功能，能够在无信号的情况下保持上一次的信号

触发器的现态和次态

  • 现态：触发器接受输入信号之前的状态
  • 次态：触发器接受输入信号之后的状态

触发器分类

按电路结构分：

  • 基本触发器：输入信号直接加到输入端，是触发器的基本电路结构，是构成其他类型触发器的基础
  • 同步触发器：输入信号经过控制门输入，控制门受时钟信号CP控制
  • 边沿触发器：只有时钟信号CP的上升沿或下降沿时刻，输入的信号才能被接收

按逻辑功能分：

  • RS触发器
  • JK触发器
  • D触发器
  • T触发器
  • T'触发器

八、计数器

计数器，用于统计输入计数脉冲CP个数的电路

原理：计数器内部由触发器构成，触发器具有记忆功能，1个触发器单元能存储1位二进制数

应用：计数、定时、分频

分类：按CP脉冲引入的方式【同步、异步计数器】、按计数值的增减趋势【加法、减法、循环计数器】、按计数容量【二进制、十进制、任意进制计数器】

8.1 同步计数器和异步计数器的区别

  • 同步计数器的外部时钟端都连在一起，而异步计数器没有
  • 同步计数器在外部信号到来时触发器同时反转；而异步计数器的触发器为串行连接，工作频率低
  • 异步计数器输出状态的建立，要比CP慢一个传输时间，容易存在竞争冒险

8.2 同步二进制计数器（74LS161）

该数码管为DCD十六进制数码管，自带译码电路


8.3 同步十进制计数器（74LS192）

8.4 任意进制计数器

通常有三种方法：

  • 直接选用已有的计数器：如欲构成十进制计数器，可选用十进制异步计数器74LS192
  • 用两个计数器串接：如用模六和模十计数器串接，构成模六十计数器
  • 利用反馈法该表原有计算器计数长度：当计数器计数到某一数值时，由电路产生置位脉冲或复位脉冲，加到计数器预置数控制端或各个触发器清零控制端，使计数器恢复到起始状态，从而达到改变计数器模的目的

8.4.1 用74LS160 反馈法构成六进制计数器

例1：反馈置零法【较优】

由此可见，N进制计数器可以利用在N-1时，将变为0的方法构成，这种方法称为反馈置零法。

例2：直接清零法

当计数器到达6时【状态6持续时间极短】，Q2、Q1均为1，使，计数器立即被强制回到零状态，开始新的循环。

  • 【以按5号为例】D5接收到一个低电平，输出为【A2A1A0 = 010】
  • 三个输出经过非门，三个T触发器分别接收到1、0、1
  • 按钮松开后，D0~D7输入均为1，优先编码器输出为全1，经过非门后，三个T触发器接收到0、0、0，故仅有第一个和第三个触发器会差生下降沿
  • 由于，刚开始时进行了清零，故三个触发器Q=0 Q逆=1，其经过四输入与非门和非门输出为1，该信号反馈给JK端，当产生CLK信号，第一个和第三个触发器发生翻转，其Q端输出为1、0、1【该信号输入进数码管译码器】
  • 【再次按其他按钮，为何不会翻转？】【以按6号为例】D6接收到一个低电平，输出为【A2A1A0 = 001】【省略若干】可以看到，经过四输入与非门和非门输出为0，T触发器处于保持状态
