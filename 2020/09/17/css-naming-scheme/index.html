以下是几种解决命名问题的方案

– BEM

– Scoped CSS

– CSS modules

一、BEM

BEM代表块（Block），元素（Element），修饰符（Modifier）。

总结

  1. BEM 让命名有规律、有含义，block 可视为模块，有一定作用域含义
  2. scoped css 限定 css 作用域，无关命名。无法适配多套主题
  3. css modules 使用算法命名，没有了命名冲突，也限定了 css 作用域。无法适配多套主题
