## 1 宽字间隔问题

问题描述：粘贴外部文字后，英文或数字的宽字间隔问题

解决：切换为半角字符

<br>

## 2 解决Word插入公式后行距不一致的情况

Word中`MathType`公式的插入，会导致原来设置为单位行距的段落行距不一致，如下左图所示。

现修改段落为五号字对应的固定行距`13.6`磅，此时行距恢复一致，但是又出现公式符号的下标被截断的情况！

![](word-1.png)

> 下表为字号对应的字高与单倍行距的磅数：

| 字号 | 字高（磅） | 单倍行距（磅） |
| :--: | :--------: | :------------: |
| 小初 |     36     |      46.6      |
| 一号 |     26     |      33.7      |
| 小一 |     24     |      31.1      |
| 二号 |     22     |      28.5      |
| 小二 |     18     |      23.3      |
| 三号 |     16     |      20.7      |
| 小三 |     15     |      19.4      |
| 四号 |     14     |      18.1      |
| 小四 |     12     |      15.5      |
| 五号 |    10.5    |      13.6      |
| 小五 |     9      |                |
| 六号 |    7.5     |                |
| 小六 |    6.5     |                |
| 七号 |    5.5     |                |



为解决公式下标截断的情况，需要调整`MathType`中公式的格式：选择符号-格式-定义间距-修改下标深度为10，即可完成修改。
