# 一 模块化编程

> [!note]
> 在早期编写 `JavaScript` 时，我们只需在 `<script>` 标签内写入 `JavaScript` 的代码就可以满足我们对页面交互的需要了。但随着时间的推移，时代的发展，原本的那种简单粗暴的编写方式所带来的诸如逻辑混乱，页面复杂，可维护性差，全局变量暴露等问题接踵而至，前辈们为了解决这些问题提出了很种的解决方案，其中之一就是 `JavaScript` 模块化编程。

介绍 `JavaScript` 的三大模块化规范：`CommonJS`、`AMD` 、`ES6`

> [!tip] 参考原文
>
> [来自简书总结，写的挺不错](https://www.jianshu.com/p/3fbaa3ae70e6/)
>
> [js-模块化(三大模块化规范)](https://www.cnblogs.com/john-hwd/p/10580620.html)

这些模块发方案有以下**四种优点**：

- 解决项目中的全局变量污染的问题。	
- 开发效率高，有利于多人协同开发。
- 职责单一，方便代码复用和维护 。
- 解决文件依赖问题，无需关注引用文件的顺序。

# 三 UI框架

UI框架是前端开发中用于快速构建美观、一致用户界面的重要工具。以下是目前主流前端框架生态中常见的UI框架：

## Vue 生态 UI框架

### 1. Element UI / Element Plus
- **简介**：由饿了么团队开发的Vue UI组件库，提供了丰富的PC端组件，设计风格简洁美观，是Vue生态中最流行的UI框架之一。
- **官网**：[Element Plus](https://element-plus.org/zh-CN/) (Vue 3版本) | [Element UI](https://element.eleme.cn/#/zh-CN) (Vue 2版本)
- **参考文档**：
  - Element Plus 官方文档：https://element-plus.org/zh-CN/component/button.html
  - Element UI 官方文档：https://element.eleme.cn/#/zh-CN/component/installation

### 2. Vuetify
- **简介**：基于Material Design设计规范的Vue UI组件库，提供了完整的Material Design风格组件，支持响应式布局。
- **官网**：[Vuetify](https://vuetifyjs.com/)
- **参考文档**：https://vuetifyjs.com/en/getting-started/quick-start/

### 3. Bootstrap Vue
- **简介**：将Bootstrap 4与Vue.js深度集成的UI组件库，提供了Bootstrap风格的组件和网格系统。
- **官网**：[Bootstrap Vue](https://bootstrap-vue.org/)
- **参考文档**：https://bootstrap-vue.org/docs

### 4. iView
- **简介**：由TalkingData团队开发的Vue UI组件库，设计风格简约现代，提供了丰富的表单、数据展示等组件。
- **官网**：[iView](https://www.iviewui.com/)
- **参考文档**：https://www.iviewui.com/components/overview

### 5. Vant
- **简介**：有赞团队开发的移动端Vue组件库，专注于移动端UI设计，提供了轻量、可靠的移动端组件。
- **官网**：[Vant](https://vant-ui.github.io/vant/#/zh-CN/)
- **参考文档**：https://vant-ui.github.io/vant/#/zh-CN/quickstart

## React 生态 UI框架

### 1. Ant Design
- **简介**：蚂蚁金服团队开发的企业级UI设计语言，提供了丰富的React组件，设计风格简洁大气，广泛应用于企业级应用。
- **官网**：[Ant Design](https://ant.design/)
- **参考文档**：https://ant.design/components/overview-cn/

### 2. Material-UI (MUI)
- **简介**：基于Material Design的React UI组件库，提供了完整的Material Design风格组件，支持主题定制。
- **官网**：[MUI](https://mui.com/)
- **参考文档**：https://mui.com/material-ui/getting-started/overview/

### 3. Ant Design Mobile
- **简介**：Ant Design的移动端版本，专注于移动端UI设计，提供了轻量、高性能的移动端组件。
- **官网**：[Ant Design Mobile](https://mobile.ant.design/)
- **参考文档**：https://mobile.ant.design/zh/guide/introduce

## 其他 UI框架

### 1. Bootstrap
- **简介**：最流行的前端UI框架之一，提供了响应式布局、组件和工具，支持多种前端框架。
- **官网**：[Bootstrap](https://getbootstrap.com/)
- **参考文档**：https://getbootstrap.com/docs/5.3/getting-started/introduction/

### 2. Tailwind CSS
- **简介**：实用优先的CSS框架，通过原子化CSS类快速构建界面，提供了高度可定制的设计系统。
- **官网**：[Tailwind CSS](https://tailwindcss.com/)
- **参考文档**：https://tailwindcss.com/docs

### 3. Semantic UI
- **简介**：语义化的UI框架，使用自然语言描述的类名，提供了丰富的组件和主题。
- **官网**：[Semantic UI](https://semantic-ui.com/)
- **参考文档**：https://semantic-ui.com/introduction/getting-started.html

## 选择UI框架的建议

1. **根据技术栈选择**：Vue项目优先考虑Element Plus、Vuetify等Vue生态UI框架，React项目优先考虑Ant Design、Material-UI等React生态UI框架。
2. **根据项目类型选择**：PC端项目推荐Element Plus、Ant Design等，移动端项目推荐Vant、Ant Design Mobile等。
3. **考虑社区活跃度**：选择社区活跃、维护更新及时的UI框架，避免使用停止维护的项目。
4. **评估学习成本**：根据团队技术背景选择学习成本适中的UI框架。
