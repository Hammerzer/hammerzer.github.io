# OpenGL 快速入门

## 学习计划

### 第一阶段：基础准备（1-2周）
- **目标**：掌握 OpenGL 开发环境配置与基础概念
- **学习内容**：
  - 配置开发环境（GLFW/GLAD/GLEW）
  - 理解 OpenGL 渲染管线
  - 学习着色器语言（GLSL）基础
  - 绘制第一个三角形

### 第二阶段：核心技术（2-3周）
- **目标**：掌握图形绘制的核心技术
- **学习内容**：
  - 纹理映射（2D/3D纹理）
  - 变换矩阵（模型/视图/投影矩阵）
  - 光照系统（Phong/Blinn-Phong光照）
  - 深度测试与模板测试

### 第三阶段：高级特性（3-4周）
- **目标**：了解 OpenGL 高级功能
- **学习内容**：
  - 帧缓冲（Framebuffer）与离屏渲染
  - 几何着色器与细分着色器
  - 阴影映射（Shadow Mapping）
  - 实例化渲染与广告牌技术

### 第四阶段：项目实践（2-3周）
- **目标**：通过项目巩固知识
- **项目建议**：
  - 3D模型加载与渲染（使用Assimp库）
  - 简单游戏引擎框架
  - 实时渲染场景（含光照与阴影）

## 必会知识点

### 1. OpenGL 基础
- **渲染管线**：顶点着色器 → 几何着色器 → 细分着色器 → 片段着色器
- **着色器语言**：GLSL语法、变量类型、精度限定符
- **VAO/VBO/EBO**：顶点数组对象、顶点缓冲对象、索引缓冲对象
- **纹理**：采样方式（线性/最近邻）、纹理坐标、多级纹理

### 2. 变换与坐标系统
- **模型矩阵**：缩放、旋转、平移
- **视图矩阵**：相机变换（LookAt矩阵）
- **投影矩阵**：正交投影、透视投影
- **坐标空间转换**：局部空间 → 世界空间 → 视图空间 → 裁剪空间 → 屏幕空间

### 3. 光照与材质
- **光照模型**：环境光、漫反射、高光
- **材质属性**：漫反射颜色、镜面反射颜色、高光强度
- **光照贴图**：漫反射贴图、法线贴图、高光贴图

### 4. 高级渲染技术
- **帧缓冲**：颜色缓冲、深度缓冲、模板缓冲
- **阴影映射**：阴影计算的基础方法
- **实例化渲染**：高效渲染大量相同物体
- **GPU粒子系统**：利用顶点着色器实现粒子效果

## 面试必考内容

### 1. 渲染管线相关问题
- **问题**：OpenGL 渲染管线分为哪几个阶段？每个阶段的主要功能是什么？
- **重点**：顶点着色器（坐标变换）、片段着色器（颜色计算）、测试阶段（深度/模板测试）

### 2. 着色器相关问题
- **问题**：GLSL 中 uniform、attribute、varying 变量的区别是什么？
- **重点**：uniform（全局常量）、attribute（顶点属性）、varying（插值传递）

### 3. 纹理映射问题
- **问题**：纹理坐标的范围是多少？如何实现纹理的重复与 clamp 效果？
- **重点**：纹理采样模式（GL_REPEAT/GL_CLAMP_TO_EDGE）

### 4. 光照与材质问题
- **问题**：Phong 光照模型由哪几部分组成？如何计算漫反射与高光？
- **重点**：法向量与光照方向的点积（漫反射）、反射光与视线方向的点积（高光）

### 5. 坐标系统问题
- **问题**：如何将一个点从世界坐标转换到屏幕坐标？
- **重点**：MVP矩阵（Model-View-Projection）的乘法顺序

## 资源推荐

- **官方文档**：[OpenGL Reference Card](https://www.khronos.org/files/opengl-quick-reference-card.pdf)
- **在线教程**：[LearnOpenGL](https://learnopengl.com/)（非常全面的中文教程）
- **书籍**：《OpenGL编程指南》（红宝书）、《OpenGL ES 3.0编程指南》
- **工具**：[Shadertoy](https://www.shadertoy.com/)（实时测试GLSL着色器）

## 学习建议

1. **实践为主**：OpenGL 是一门实践性很强的技术，需要大量的代码练习
2. **理解原理**：不要只记代码，要理解每个步骤的原理
3. **参考优秀代码**：阅读开源项目（如 [LearnOpenGL 示例](https://github.com/JoeyDeVries/LearnOpenGL)）
4. **持续学习**：OpenGL 版本不断更新，关注最新特性（如 Vulkan、Metal）
